Village Map

Villages
Load a 690x690 map image to begin.
Manual placement: left-click to place a town. A town touching an existing town merges into it (odd shapes allowed). Right-click erases the squares it overlaps; a town with no squares left is removed.

Biome labels

Label each map color. Use the word "water" for impassable water biomes.

Tier labels

Label each tier color. Villages may only be placed on tiers whose name contains 1, 2 or 3.

Guilds

Towns assigned to a guild are drawn in the guild color. Add a guild above, then select it before placing or spawning towns.
